# プライバシーポリシー

FoodGram（以下，「当社」といいます。）は，本アプリ上で提供するサービス（以下,「本サービス」といいます。）における，ユーザーの個人情報の取扱いについて，以下のとおりプライバシーポリシー（以下，「本ポリシー」といいます。）を定めます。

## 本ポリシーの要点

- 位置情報は，ユーザーが許可した場合にのみ取得します
- マップ表示・レストラン検索のために現在地を利用しますが，現在地そのものを当社データベースに保存しません
- 投稿で取り扱う位置情報は，レストランの位置情報のみです
- 投稿画像は他のユーザーに公開されますが，画像メタデータを当社が個別項目として抽出・保存する設計にはしていません
- 個人情報は，本サービス運営に必要な範囲でのみ外部サービス事業者に提供します
- 当社は，ユーザー体験を損なう過度な広告配信を行わない方針です
- 投稿・プロフィール・アカウントは，アプリ内操作で削除または変更できます（一部はお問い合わせ対応）

## 第1条（個人情報）

「個人情報」とは，個人情報保護法にいう「個人情報」を指すものとし，生存する個人に関する情報であって，当該情報に含まれる氏名，生年月日，住所，電話番号，連絡先その他の記述等により特定の個人を識別できる情報及び容貌，指紋，声紋にかかるデータ，及び健康保険証の保険者番号などの当該情報単体から特定の個人を識別できる情報（個人識別情報）を指します。

当社が収集する個人情報は以下のとおりです。

1. **認証情報**
   - ユーザーID（Supabase認証）
   - メールアドレス（Google/Apple認証を使用する場合）

2. **プロフィール情報**
   - 名前
   - ユーザー名
   - 自己紹介文
   - プロフィール画像

3. **投稿情報**
   - 投稿した食べ物の画像
   - 食べ物の名前
   - コメント
   - レストラン情報
   - 位置情報（緯度・経度）
   - 画像ファイル（画像に付随する情報が含まれる場合があります。詳細は第8条）

4. **利用状況情報**
   - アプリの利用履歴
   - プレミアム会員情報
   - 投稿日時、更新日時

5. **デバイス情報**
   - デバイスの種類、OS情報（エラーレポート等の目的で使用）

## 第2条（個人情報の利用目的）

当社は，取得した個人情報を以下の目的で利用します。

1. 本サービスの提供・運営のため
2. ユーザーからのお問い合わせへの対応のため
3. ユーザーに重要なお知らせを配信するため
4. メンテナンス，重要なお知らせなど必要に応じたご連絡のため
5. 利用規約に違反したユーザーや，不正・不当な目的でサービスを利用しようとするユーザーの特定をし，ご利用をお断りするため
6. ユーザーにご自身の登録情報の閲覧・変更・削除・ご利用状況の閲覧を行っていただくため
7. 上記の利用目的に付随する目的

## 第3条（利用目的の変更）

当社は，利用目的が変更前と関連性を有すると合理的に認められる場合に限り，個人情報の利用目的を変更するものとします。利用目的の変更を行った場合には，変更後の目的について，当社所定の方法により，ユーザーに通知し，または本アプリ上に公表するものとします。

## 第4条（個人情報の第三者提供）

当社は，次に掲げる場合を除いて，あらかじめユーザーの同意を得ることなく，第三者に個人情報を提供することはありません。ただし，個人情報保護法その他の法令で認められる場合を除きます。

1. 人の生命，身体または財産の保護のために必要がある場合であって，本人の同意を得ることが困難であるとき
2. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって，本人の同意を得ることが困難であるとき
3. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって，本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき
4. 予め次の事項を告知あるいは公表し，かつ当社が個人情報保護委員会に届出をしたとき
   1. 利用目的に第三者への提供を含むこと
   2. 第三者に提供されるデータの項目
   3. 第三者への提供の手段または方法
   4. 本人の求めに応じて個人情報の第三者への提供を停止すること
   5. 本人の求めを受け付ける方法

前項の定めにかかわらず，次に掲げる場合には，当該情報の提供先は第三者に該当しないものとします。

1. 当社が利用目的の達成に必要な範囲内において個人情報の取扱いの全部または一部を委託する場合
2. 合併その他の事由による事業の承継に伴って個人情報が提供される場合
3. 個人情報を特定の者との間で共同して利用する場合であって，その旨並びに共同して利用される個人情報の項目，共同して利用する者の範囲，利用する者の利用目的および当該個人情報の管理について責任を有する者の氏名または名称について，あらかじめ本人に通知し，または本人が容易に知り得る状態に置いた場合

当社は，本サービスの提供にあたり，以下の第三者サービスを利用しており，本サービスの提供に必要な範囲で個人情報を提供する場合があります。

1. **Supabase**（データベース・ストレージサービス）
   - 提供する情報：ユーザー情報、投稿情報、画像データ
   - 利用目的：データの保存・管理

2. **Firebase**（プッシュ通知サービス）
   - 提供する情報：ユーザーID、デバイストークン
   - 利用目的：プッシュ通知の配信

3. **RevenueCat**（サブスクリプション管理サービス）
   - 提供する情報：ユーザーID、購入情報
   - 利用目的：プレミアム会員の管理

4. **Google Mobile Ads**（広告配信サービス）
   - 提供する情報：広告ID、デバイス情報（ユーザーの同意がある場合）
   - 利用目的：広告の配信、広告効果の測定

これらのサービス提供者は，それぞれのプライバシーポリシーに従って個人情報を取り扱います。

また，当社は，以下を目的とした個人情報の販売・名寄せ・提供は行いません。
- 本サービスの提供に不要な目的での第三者提供
- ユーザーの同意なく行う広告目的での個人情報の外部提供（法令で認められる場合を除きます）

第三者提供の考え方をわかりやすく示すため，本サービスにおける主な公開・提供範囲を以下に定めます。

1. **他のユーザーに公開される情報（投稿時）**
   - 投稿画像、食べ物名、コメント、レストラン情報
   - 投稿に付随するレストランの位置情報（ユーザーが投稿に含めた場合）

2. **外部のサービス事業者に提供される情報（本サービス運営上必要な範囲）**
   - データ保存・通知配信・サブスクリプション管理・広告配信に必要な情報

3. **公開も提供もしない情報（原則）**
   - パスワードそのもの（認証基盤側で管理）
   - 本サービス提供に不要な個人情報

## 第5条（個人情報の訂正および削除）

ユーザーは，当社の保有する自己の個人情報が誤った情報である場合には，当社が定める手続きにより，当社に対して個人情報の訂正，追加または削除（以下，「訂正等」といいます。）を請求することができます。

当社は，ユーザーから前項の請求を受けてその請求に応じる必要があると判断した場合には，遅滞なく，当該個人情報の訂正等を行うものとします。

当社は，前項の規定に基づき訂正等を行った場合，または訂正等を行わない旨の決定をしたときは遅滞なく，これをユーザーに通知します。

## 第6条（個人情報の利用停止等）

当社は，本人から，個人情報が，利用目的の範囲を超えて取り扱われているという理由，または不正の手段により取得されたものであるという理由により，その利用の停止または消去（以下，「利用停止等」といいます。）を求められた場合には，遅滞なく必要な調査を行います。

前項の調査結果に基づき，その請求に応じる必要があると判断した場合には，遅滞なく，当該個人情報の利用停止等を行います。

当社は，前項の規定に基づき利用停止等を行った場合，または利用停止等を行わない旨の決定をしたときは，遅滞なく，これをユーザーに通知します。

前2項にかかわらず，利用停止等に多額の費用を有する場合その他利用停止等を行うことが困難な場合であって，ユーザーの権利利益を保護するために必要なこれに代わるべき措置をとれる場合は，この代替策を講じるものとします。

## 第7条（位置情報の取り扱い）

当社は，本サービスの提供にあたり，ユーザーの位置情報（GPS情報）を取得する場合があります。ユーザーの明示的な許可がない限り，位置情報を取得しません。

1. **取得する位置情報**
   - マップ表示・レストラン検索時の現在地の緯度・経度（端末上で利用）
   - 投稿に利用されるレストランの位置情報（緯度・経度）

2. **利用目的**
   - レストランの位置情報をマップ上に表示するため
   - 近くのレストランを検索するため
   - 位置情報は，ユーザーの安全に配慮し，必要に応じて一定の精度で表示される場合があります

3. **取得方法**
   - ユーザーが明示的に許可した場合のみ取得します
   - 位置情報の取得は任意です（位置情報を許可しない場合でも，本サービスの基本的な機能はご利用いただけます）
   - 取得した現在地は，マップ表示およびレストラン検索のために利用し，当社データベースには保存しません
   - 投稿時に保存される位置情報は，ユーザーが投稿に設定したレストラン位置情報です

4. **第三者への提供**
   - 現在地の位置情報は，投稿に含めない限り，第三者に提供しません
   - 投稿に含まれるレストラン位置情報は，他のユーザーに公開されます
   - ユーザー個人の現在地や行動履歴を，特定できる形で公開することはありません

5. **ユーザーによる停止・削除**
   - 端末の設定で，本アプリの位置情報権限をいつでも停止できます
   - 投稿済みの位置情報は，該当投稿の編集または削除により取り除くことができます

## 第8条（写真データの取り扱い）

当社は，ユーザーが投稿またはプロフィール設定のために選択した写真データを，サービス提供のために取り扱います。

1. **取得する写真データ**
   - ユーザーが投稿した画像
   - ユーザーが設定したプロフィール画像
   - 画像ファイル内にメタデータ（例：撮影日時・位置情報等）が含まれる場合があります
   - 当社は，これらのメタデータを個別の項目として抽出・保存する設計にはしていません
   - 当アプリは，画像アップロード時に原則としてメタデータ（位置情報等）を削除しますが，端末や処理環境により一部が残る可能性があります

2. **利用目的**
   - 投稿・プロフィール表示機能の提供
   - 画像の保存・配信・バックアップ
   - 不正利用や規約違反への対応

3. **公開範囲**
   - 投稿画像は，他のユーザーが閲覧できる状態で公開されます
   - プロフィール画像は，プロフィール情報として他のユーザーに表示されます

4. **ユーザーが注意できること**
   - 写真に位置情報等のメタデータが含まれる場合があります。公開前に，端末側でメタデータ削除等をご検討ください
   - 公開済み画像は，投稿削除後もバックアップ等から一定期間復元される可能性があります（法令または運用上必要な場合）

5. **削除方法**
   - 投稿画像：該当投稿を削除することで削除できます
   - プロフィール画像：プロフィール編集で変更または削除できます

## 第9条（個人情報の保存期間・保存場所）

1. **保存期間**
   - 当社は，ユーザーがアカウントを削除するまで，または本サービスの提供を終了するまで，個人情報を保存します
   - アカウント削除後は，法令で定められた保存期間を除き，原則として即時に削除します

2. **保存場所**
   - 個人情報は，Supabaseのサーバー（主に日本国内）に保存されます
   - Firebase、RevenueCat、Google Mobile Adsでは，本サービス運営に必要な識別子等が各社サービス上で処理・保存される場合があります

## 第10条（削除方法）

ユーザーは，アプリ上の操作またはお問い合わせにより，ご自身の情報の削除を求めることができます。

1. **投稿データの削除**
   - 投稿画面または投稿詳細画面から，ユーザー自身で投稿を削除できます

2. **プロフィール情報の削除・変更**
   - プロフィール編集画面から，名前・自己紹介・プロフィール画像等を変更または削除できます

3. **アカウント削除**
   - アプリ内の設定画面等から退会手続きを行うことで，アカウントを削除できます
   - アカウント削除後は，法令で保存が義務付けられる情報を除き，原則として即時に削除します

4. **お問い合わせによる削除依頼**
   - アプリ操作で削除できない情報がある場合は，第13条のお問い合わせ窓口までご連絡ください
   - ご本人確認のうえ，法令に基づき対応可否を回答します

## 第11条（広告トラッキング）

当社は，本サービスにおいてGoogle Mobile Adsを使用して広告を配信しています。

1. **トラッキング技術の使用**
   - 広告の配信と効果測定のため，トラッキング技術（広告ID等）を使用する場合があります
   - iOS 14.5以降では，App Tracking Transparency（ATT）により，ユーザーの明示的な同意を得た場合のみトラッキングを行います

2. **ユーザーの権利**
   - ユーザーは，設定画面からトラッキングの許可・拒否を選択できます
   - トラッキングを拒否した場合でも，本サービスの基本的な機能はご利用いただけます（広告は引き続き表示されますが，パーソナライズされない広告となります）
   - 当社は，ユーザー体験を損なう過度な広告配信を行わない方針です

## 第12条（プライバシーポリシーの変更）

本ポリシーの内容は，法令その他本ポリシーに別段の定めのある事項を除いて，ユーザーに通知することなく，変更することができるものとします。

当社が別途定める場合を除いて，変更後のプライバシーポリシーは，本ウェブサイトに掲載したときから効力を生じるものとします。
