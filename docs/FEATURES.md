# Features

This document summarizes the main features of Food Gram App.

## Authentication

### Sign-In Methods
- **Google Sign-In**: Sign in with Google account
- **Apple Sign-In**: Sign in with Apple ID (iOS)
- **New Account Creation**: Register with email and password

### Authentication Flow
- Splash screen → Tutorial → Authentication screen
- Tutorial is displayed on first launch
- Authenticated users are automatically logged in

## Posting Features

### Post Creation
- **Photo Posting**: Post multiple photos
- **Location Information**: Automatically obtained via GPS or manually entered
- **Tagging**: Categorize dishes by type and characteristics using tags
- **Description**: Add descriptions to posts

### Post Management
- **Post Editing**: Edit your own posts
- **Post Deletion**: Delete your own posts
- **Post Details**: Display detailed information about posts

### Post Display
- **Timeline**: Display latest posts in chronological order
- **Profile**: Post list for each user
- **Search**: Search posts by keywords or tags

## Social Features

### Engagement
- **Like**: Like posts
- **Share**: Share posts to external apps

### Follow Functionality
- **Follow/Followers**: Follow other users
- **Profile Viewing**: View other users' profiles

## Map Features

### Map Display
- **Interactive Map**: Display post location information on the map
- **Pin Display**: Display pins at post locations
- **Map Style**: Use custom map styles

### Location Services
- **Current Location Display**: Display user's current location
- **Nearby Restaurants**: Get restaurants near current location

## Profile Features

### User Profile
- **Profile Image**: Set custom profile image
- **Post List**: Display list of your posts
- **Statistics**: Display post count, like count, etc.

### Profile Editing
- **Basic Information Editing**: Edit username, bio, etc.
- **Profile Image Change**: Change profile image

## Notification Features

### Push Notifications
- **Like Notifications**: Notify when your post receives a like

### Notification Settings
- **Notification ON/OFF**: Enable/disable notifications

## Premium Features

### Subscription
- **Premium Membership**: Subscription using RevenueCat
- **Premium Benefits**: Ad-free experience, access to special features, etc.

### Payment Management
- **Subscription Management**: Manage subscriptions within the app
- **Restore**: Restore purchase history

## Advertising Features

### Ad Display
- **Banner Ads**: Display banner ads at the bottom of the screen
- **Interstitial Ads**: Display full-screen ads during screen transitions
- **Open Ads**: Display ads when the app launches

### Ad Settings
- **Ad Tracking Permission**: Support for ATT (App Tracking Transparency) on iOS 14.5+

## Other Features

### Multi-language Support
- **Supported Languages**: Japanese, English, Korean, Chinese, Thai, Vietnamese, etc.
- **Auto-switching**: Automatically switch based on device language settings

### App Settings
- **Notification Settings**: Detailed notification settings
- **Account Management**: Account deletion, etc.

### Force Update
- **Version Check**: Check for latest version on app launch
- **Update Prompt**: Prompt for update if a mandatory update is available
